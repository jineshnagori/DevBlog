My First Contribution for Hacktoberfest 2025

Posted on October 4, 2025 by avadhutmali
Hacktoberfest Open Source

Contributing to open source for the first time can seem intimidating, but Hacktoberfest makes it really accessible. I decided to contribute to the DevBlog project because it was listed as beginner-friendly.

The process involved forking the repository, cloning it to my computer, and setting up the project in VS Code. The biggest step was creating a new HTML file for my blog post and using the provided template to get started. After writing my content, I had to add a link to my new post on the homepage.

Finally, I committed my changes and created a Pull Request. It's a great feeling to see your work submitted and to know you've become a part of a larger project. I'm excited to see my profile show up on the contributors page!
